# 💻 Código Fuente
Esta carpeta contiene todos los archivos de programación del proyecto.

Subid vuestro código aquí manteniendo la estructura de paquetes".
Organizar en carpetas los trabajos y cada trabajo o reto lo podeis dividir en subcarpetas :
Ejemplos
  .- Capturas y Sensores
  .- Conversiones envío
  .- Transmisión y payloads
  .- Recepción y conversión
  .- Captura y envio a cuadros de mando
  .- Cuadros de mando y analisis datos
  .- Alarmas y sistemas de control

  *Como lo se estructura en carpetas puede ser que no se precisen todas o que fuera necesario incorporar alguna mas que no este aqui encuadrada. 


### 📝 Normas de esta carpeta:
* **Organización:** Cread una subcarpeta con un nombre precedido por la iniciales del ciclo (STI/EM) si vais a subir módulos individuales.
* **Calidad:** El código debe estar comentado y seguir las normas de estilo explicadas en clase.
* **Limpieza:** Antes de subir (commit), aseguraos de que el código compila y funciona.
