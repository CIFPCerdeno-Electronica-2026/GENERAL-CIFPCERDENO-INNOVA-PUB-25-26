# 🚀 Proyecto de INNOVACIO EN EL METAVERSO - Curdo 2025-26

## ** Colaboración intercentros: Gran Canaria, Málaga y Asturias**

Bienvenido al repositorio público del equipo de proyectos especiales de **Educastur en el Principado Asturias/ CIFP Cerdeño**. En este espacio, los alumnos aplican sus conocimientos para resolver retos reales de IoT.

---

## 🎯 Objetivo del Proyecto
Este repositorio contiene el desarrollo de diferentes soluciones en ambitos diversos desde industrial, salud, o cualquier otro ambito que requiera control y automátización de sistemas.
Las soluciones técnicas para [Creación de arquitecturas de comunicaciones para la gestión de datos de control de entornos ambientales o perimetrales o salud por ejemplo]. 

El objetivo es fomentar el **trabajo colaborativo**, el uso de **buenas prácticas de código** y el aprendizaje de la cultura **Open Source**.

## 🛠️ Tecnologías Utilizadas
En este proyecto estamos trabajando con:
* **Lenguaje:** Python / C++
* **Arquitecturas:** Microcontroladores (Arduino , Rasperri pi pico, ESP32, STM32, etc )
* **Arquitecturas 2:** SOC (Raspberry pi 5,  Arduino Uno Q, pequeños portatiles)
* **Protocolos y herramientas IOT** MQTT, Node-RED, Influxdb, Grafana, etc...
* **Entornos operativos :** Hypervisores y contenedores ( VBOX, VMWARE , DOCKER)
* **Entornos Simulación:** WOKWI, Google Cloud Shell etc..
* **Herramientas IDE:** VScode, Platformio, Thonny
* **Herramientas edición evidencias** Portatil y tarjeta capturadora con camaras de video y sistemas de iluminación
* **Soportes soluciones** Entornos 3D de empresa WION y servidores de empresa Tecnogestión
* **Soporte Formación IoT** Empresa Aritium 
* **Control de versiones:** Git & GitHub
* **Licencia:** MIT (Open Source)

## 📂 Estructura del Repositorio
Para mantener el orden, los alumnos deben seguir esta estructura de carpetas:
* `/docs`: Documentación pública y diagramas.
* `/src`: Código fuente del proyecto.
* `/tests`: Pruebas de funcionamiento.
* `/assets`: Imágenes y recursos gráficos.

Cualquier otra información que no corresponda a estas carpetas busca la alternativa en el repositorio de trainnig o pruebas interno

## ⚠️ Normas de Contribución para Alumnos
Si eres alumno de este proyecto, recuerda:
1. **Seguridad:** Nunca subas contraseñas, claves API o datos personales al código.
2. **Mensajes de Commit:** Usa mensajes descriptivos (ej: `feat: añade validación de formulario`).
3. **Respeto:** Este es un escaparate público de tu trabajo. Mantén un código limpio y comentarios profesionales.
